# 🌍 MundoCraft

Um projeto moderno e modular construído com Vite, React, TypeScript e TailwindCSS — idealizado para performance, escalabilidade e design elegante.

[🔗 Acesse o projeto publicado via GitHub Pages](https://harrisalexandre.github.io/mundocraft/index)

---

## 🚀 Tecnologias Utilizadas

- **Vite** — bundler ultra-rápido para aplicações modernas
- **React com SWC** — performance otimizada na renderização
- **TypeScript** — segurança e clareza no desenvolvimento
- **TailwindCSS** — estilização rápida, responsiva e reutilizável
- **Radix UI** — componentes acessíveis e sofisticados
- **React Hook Form + Zod** — validações tipadas e eficientes
- **React Router Dom** — navegação client-side fluida
- **Recharts** — gráficos interativos e elegantes
- **TanStack Query (React Query)** — gerenciamento de dados remoto

---

## 🧠 Arquitetura

- 🔌 **Componentização inteligente** com uso de `@/` via alias
- 💅 Estilo baseado em variáveis, tokens e animações utilitárias
- 📁 Organização modular (por domínio/feature)
- 🎯 Código preparado para CI/CD e deploy automatizado

---

## 📦 Instalação local

```bash
git clone https://github.com/seuusuario/mundocraft.git
cd mundocraft
npm install
npm run dev
```
